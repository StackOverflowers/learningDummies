    Learning Dummies es un sitio web que ofrece los mejores tips de programación para usuarios novatos.

    Puedes encontrar temas como:

    • HTML que significa HyperText Markup Language, es decir, hace referencia al lenguaje de marcado para la elaboración de páginas web. Su versión más reciente es la 5.2.
    • CSS en español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.​ Su última versión es la 3.
    • JavaScript el cual es uno de los lenguajes de programación más usados actualmente, es interpretado y viende del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico. Su última versioón estable es LA ECMAScript 2016.
    • Bootstrap que es una biblioteca multiplataforma o conjunto de herramientas de código abierto para diseño de sitios y aplicaciones web.
    • También puedes aprender sobre lógica , que es la técnica utilizada para desarrollar instrucciones en una secuencia para lograr un objetivo. En otras palabras es la organización y planificación de instrucciones en un algoritmo, con el objetivo de tornar visible la implementación de un programa o software.
    • Es muy importante que aprendas sobre repositorios de código, aquí encontrarás información sobre cómo guardar el código de una aplicación.
    • Finalmente, para que puedas trabajar en equipo adecuadamente aprenderás en Learning Dummies sobre SCRUM que es un proceso de gestión que reduce la complejidad en el desarrollo de productos para satisfacer las necesidades de los clientes.

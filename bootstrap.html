    ¿Que es Bootstrap 5?

    Por:Juan Manuel Huertas  ;  Fecha: Jueves 21 de julio

    Es una nueva versión de Bootstrap que elimina completamente JQuery, para trabajar sobre Javascript puro, y trae varias características como:

    * Mejoría del sistema de cuadrícula
    * Documentación mas clara y completamente
    *Modularidad mejorada
    *Personalización mas simple

    ¿Como se instala Bootstrap 5?

    Por:Juan Manuel Huertas  ;  Fecha: Jueves 21 de julio

    Debes colocar en el Header, la siguiente linea:
